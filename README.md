### 01. 🌶️ Spice Garden — Restaurant Website
A premium Indian restaurant website with full menu display, table reservation system, customer reviews, and smooth scroll animations.

**Features:**
- Interactive menu with category filters
- Table reservation form
- Customer testimonials section
- Animated hero section
- Stats counter display

- Mobile responsive design

## 🔧 Tech Stack

```
Frontend:
├── HTML5          — Semantic structure
├── CSS3           — Styling & animations
└── JavaScript     — Interactivity & DOM manipulation

Features Used:
├── CSS Grid & Flexbox   — Responsive layouts
├── CSS Animations       — Smooth transitions

└── CSS Variables        — Consistent theming
```
